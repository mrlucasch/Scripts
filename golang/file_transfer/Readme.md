

# File Transfer Program


The client reads a copy of alice in wonderland(167kb) and then sends it over and over to the server until 10Gb have been transferred.

## Building

```
make all
```
or 
```
make server
```
or 
```
make client
```

## Running
On the server run:
```
make server
```


On the client run:
```
make test
```
